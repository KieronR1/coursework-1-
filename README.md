Craigslist Rebrand - Client-side Development ProjectProject OverviewThis project is a modern rebranding of the Craigslist platform,
created for the 4COSC012W Client-side Development Fundamentals module. The aim is to transform the existing text-heavy, outdated interface into a visually engaging, interactive,
and responsive user experience.Key FeaturesModern UI/UX: Replaced the legacy layout with a clean, grid-based card system for listings.
Dynamic Hero Section: Integrated a particle-based background using tsParticles to create a cutting-edge first impression.Responsive Design: Implemented media queries to ensure the site scales 
smoothly from desktop to mobile devices.Brand Consistency: Used a dedicated color palette (Purple, White, Grey) and modern sans-serif typography to establish trust and reliability.
Technical StackHTML5: Semantic structure for accessibility and SEO.CSS3: Custom properties (variables), Flexbox layouts, and CSS transitions for interactive "pop" effects.
JavaScript: Used for DOM manipulation and initializing the particle engine.Libraries: tsParticles for the dynamic background.Project StructureThe repository is organized into a modular file 
structure to ensure maintainability:index.html: The main entry point for the website.style.css: All layout and styling rules, including responsive breakpoints.
C.L2.js: JavaScript logic for asset initialization and interactivity./images: A dedicated directory for optimized static assets like the logo, product photos, and SVG icons.
